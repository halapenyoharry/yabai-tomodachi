## Thread Posts

**1/**
Just launched yabai-tomodachi - a friendly companion app for Yabai window management on macOS! 🪟✨

It adds the menu bar controls and AI integration that Yabai users have been asking for.

🧵👇

**2/**
If you use Yabai, you know the pain:
- Windows get "stuck" → need to restart
- Forget the exact command syntax
- Want to arrange windows but can't remember how

yabai-tomodachi solves all of these.

**3/**
Features:
🖱️ Menu bar for instant access to controls
🤖 AI integration - tell Claude what you want
🔄 One-click Yabai restart (most requested!)
📦 Beautiful installer with Yabai setup help
🎨 Cute icon because developer tools should spark joy

**4/**
The coolest part? Natural language window control through Claude:

"Set up my coding workspace"
"Move all browsers to space 2"  
"Balance the windows on this screen"

No more memorizing commands!

**5/**
Built with:
- Swift for the native menu bar app
- TypeScript for the MCP (Model Context Protocol) server
- Love for the Yabai community

100% open source: github.com/halapenyoharry/yabai-tomodachi

**6/**
This is part of a bigger vision: What if AI tool protocols weren't just for AI?

MCP could become a universal standard for tool discovery and control. Any app could use any tool. The menu bar app is just the beginning.

**7/**
